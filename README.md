# Dr. Rajesh Varma — Literary Website

Next.js (React) site for Dr. Rajesh Varma's literary portfolio.

## Local development

```bash
npm install --prefix frontend
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy on Vercel

1. Import this repo at [vercel.com/new](https://vercel.com/new)
2. Set **Root Directory** to `frontend` (recommended), or use the root `vercel.json`
3. Deploy — Vercel auto-detects Next.js

## Routes

- `/` — Home
- `/about` — Biography
- `/books` — Literary works
- `/awards/sahitya`, `/awards/booker`, `/awards/jnanpith`, `/awards/padma`
